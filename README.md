# Vrinda-Store-MS-Excel-Analysis
Conducted a retail sales analysis project using Microsoft Excel to evaluate customer behavior and sales trends. Performed data cleaning, transformation, and analysis using Pivot Tables. Developed an interactive dashboard with charts and slicers to present insights.

# Tools Used

The project was carried out using:
- Microsoft Excel for data cleaning, preprocessing, and initial analysis
- Pivot Tables and Charts for summarizing data
- Dashboard Features in Excel for visualization and reporting

# Data Collection
The dataset was obtained in a structured spreadsheet format containing transactional and customer-related information for the store. The data included fields such as order details, customer demographics, sales channels, product categories, and order status.

# Data Cleaning

The raw dataset was preprocessed to ensure accuracy and consistency before analysis. The following steps were performed:
- Removed duplicate records to avoid redundancy
- Handled missing and blank values by either correcting or excluding them
- Standardized text entries (such as gender, categories, and status) to maintain uniformity
- Corrected inconsistent formatting in date and categorical fields
- Verified data types to ensure proper classification of numerical and text fields
- Data Transformation

After cleaning, the dataset was transformed to make it suitable for analysis:
- Created new columns such as month extraction from dates for time-based analysis
- Grouped similar categories to simplify interpretation
- Converted raw data into structured formats using tables
- Applied filters and sorting techniques to segment the dataset

# Data Analysis

The cleaned data was analyzed using Excel features:

- Pivot Tables were used to summarize sales performance across different dimensions such as time, category, and customer segments
- Customer behavior was analyzed based on demographics and purchasing patterns
- Sales performance was evaluated across different sales channels and regions
- Order status analysis was conducted to understand fulfillment trends

# Data Visualization

An interactive dashboard was created to present insights clearly:
- Used bar charts, pie charts, and line charts to represent trends and comparisons
- Applied slicers and filters to make the dashboard interactive
- Organized visuals in a structured layout for easy understanding
- Highlighted key performance indicators for quick insights

# Insights Generation

Based on the analysis:
- Identified patterns in customer purchasing behavior
- Observed trends across different time periods
- Compared performance across various product categories and sales channels
- Evaluated order fulfillment and operational efficiency

# Conclusion
The project demonstrates how raw business data can be transformed into meaningful insights using Excel. The use of data cleaning, transformation, and visualization techniques helped in understanding sales trends and customer behavior, enabling better decision-making.
